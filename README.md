🚀 PoLE – Proof of Learning Effort
📌 Overview

PoLE (Proof of Learning Effort) is a Blockchain-Based Learning Effort Proof System built during the 24-hour Code Kalari Hackathon at IIIT Kottayam.

The project addresses the growing issue of fake and unverifiable student credentials by combining AI + Web3 technologies to validate learning authenticity before storing records on the blockchain.

🎯 Problem Statement

Fake certificates and unverifiable credentials make it difficult for HRs and institutions to assess genuine skills. Traditional resumes lack proof of actual learning effort.

💡 Solution

PoLE verifies credentials through a multi-step validation process:

📄 Students upload:

Project details

Hackathon participation records

Online course certificates

🔍 OCR + AI Analysis:

Extracts text using Tesseract OCR

Validates content using LLMs

🧠 Knowledge-Based Quiz:

System generates a quiz based on uploaded content

Evaluates performance

Assigns a credibility grade

🔗 Blockchain Verification:

Only verified credentials are stored on Ethereum Smart Contracts

Generates a tamper-proof QR Code

📲 Instant Verification:

HRs can scan the QR code to view immutable student records

🏗️ Tech Stack

Frontend: React (Vite)

Backend: Node.js

AI Layer: Python, Tesseract OCR, LLMs

Blockchain: Ethereum Smart Contracts

Wallet Integration: MetaMask

Deployment/Infra: Kaava.io

⚙️ Architecture Flow

Upload → OCR → AI Validation → Quiz Generation → Grading → Smart Contract Storage → QR Generation → HR Verification

AI-based credential verification
